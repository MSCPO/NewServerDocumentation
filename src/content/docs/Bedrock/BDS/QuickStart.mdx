## 下载 BDS

<Steps>

1. 访问 [Minecraft 官方下载页](https://www.minecraft.net/zh-hans/download/server/bedrock) 下载 BDS
2. 下载完成后解压到任意目录

</Steps>

## 启动服务端

<Tabs syncKey="os">
<TabItem label="Windows">

双击 `bedrock_server.exe` 即可启动服务端

</TabItem>
<TabItem label="Linux">

在服务端目录下打开终端，并执行命令 `./bedrock_server` 即可启动服务端

:::caution
Bedrock 版专属服务器的 Linux 版本需要 Ubuntu 20.04（LTS 版本）或更高版本，不支持其他发行版。
:::

</TabItem>
</Tabs>
